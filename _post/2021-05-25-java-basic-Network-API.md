
## java.net API

- IP 관련 : InetAddress

- Web 관련 : URL

- TCP 관련 : Socket, ServerSocket


### InetAddress

- IP 정보를 받아올 때 사용하는 클래스

- 공식문서 : https://docs.oracle.com/javase/8/docs/api/java/net/InetAddress.html

- 인스턴스 객체를 만들 필요가 없다. 생성자가 없다. static 메소드를 이용해 접근하는 것이다.
